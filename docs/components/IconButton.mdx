## Usage
```tsx
import React from 'react';
import {IconButton} from './components';

const Example = () => {
  return (
    <IconButton
      variant="vector"
      icon="bell"
      iconStyle="contained"
      color="danger"
    />
  );
};

export default Example;
```

## Props

IconButton extends [`...Icon Prop`](/guides/content/editing-an-existing-page#modifying-front-matter) from the Icon component


### `iconStyle`



**Type:** `'contained' | 'outlined'`

- contained : Icon with a solid rounded background.

- contained : Icon with a rounded border.


### onPress

**Type:**  `(e: GestureResponderEvent) => void`

Function to execute on press.


### disabled

**Type:**  `booealn`

Determines the iconbutton is disabled.
