A Switch is a visual control that toggles between two opposite states: on and off.

## Usage

```tsx
import React from 'react';
import { Switch } from './components';

const Example = () => {
  return <Switch thumbColor="primary" value={true} />;
};

export default Example;
```
## Props

Extends the [`SwitchProps`](https://reactnative.dev/docs/switch#props) from React Native.

### trackColor

**Type:** `string`

### thumbColor

**Type:** `string`

### ios_backgroundColor

**Type:** `string`
